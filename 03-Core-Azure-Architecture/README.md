# 🏗️ Core Azure Architecture
 
This section covers the fundamental components of Microsoft Azure and how resources are structured and managed.
 
---
 
## 📚 Topics
 
- Regions  
- Availability Zones  
- Resource Groups  
- Subscriptions  
- Management Groups  
 
---
 
## 🧠 Azure Hierarchy
 
Management Group  
↓  
Subscription  
↓  
Resource Group  
↓  
Resources  
 
---
 
## 🌍 Infrastructure Hierarchy
 
Region  
↓  
Availability Zones  
↓  
Data Centers  
 
---
 
## 🎯 Key Points
 
- Azure resources are organized in a hierarchical structure  
- Resource Groups are used to manage related resources  
- Subscriptions handle billing and access control  
- Management Groups provide governance across subscriptions  
- Regions and Availability Zones ensure availability and performance
