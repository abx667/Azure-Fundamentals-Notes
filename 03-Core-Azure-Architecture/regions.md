# 🌍 Azure Regions
 
An Azure region is a geographical area that contains one or more data centers.
 
---
 
## 🔑 Key Points
 
- Each region is made up of multiple data centers  
- Services are deployed within a region  
- Helps in reducing latency (closer to users)  
- Provides data residency and compliance  
 
---
 
## 📌 Example
 
India Central, East US, West Europe
 
---
 
## 🎯 Important
 
- Choose region based on:
  - Location of users  
  - Cost  
  - Compliance requirements  
 
---

- Region = geographic location
