# 📦 Resource Groups
 
A resource group is a logical container that holds related Azure resources.
 
---
 
## 🔑 Key Points
 
- All resources must belong to a resource group  
- Used to organize and manage resources  
- Enables monitoring, access control, and lifecycle management  
 
---
 
## 📌 Example
 
A web application, database, and storage can be grouped together.
 
---
 
## 🎯 Important
 
- Resources in a group can be in different regions  
- Deleting a resource group deletes all resources inside it  
 
---
 
## 🧠 Quick Revision
 
- Resource Group = container for resources  
- Used for management and organization
