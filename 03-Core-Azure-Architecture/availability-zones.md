# 🏢 Availability Zones
 
Availability Zones are physically separate data centers within a region.
 
---
 
## 🔑 Key Points
 
- Each zone has independent power, cooling, and networking  
- Protects applications from data center failures  
- Enables high availability  
 
---
 
## 📌 Example
 
A region may have 3 availability zones (Zone 1, Zone 2, Zone 3)
 
---
 
## 🎯 Important
 
- If one zone fails → others continue working  
- Used for mission-critical applications  
 
---
 
## 🧠 Quick Revision
 
- Zones = separate data centers inside a region
