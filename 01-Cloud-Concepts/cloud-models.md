# ☁️ Cloud Deployment Models
 
---
 
## 🌍 Public Cloud
- Owned by cloud provider
- Shared resources
- Example: Microsoft Azure
 
---
 
## 🔒 Private Cloud
- Used by one organization only
- More control and security
 
---
 
## 🔄 Hybrid Cloud
- Combination of public + private cloud
- Most commonly used in real-world
