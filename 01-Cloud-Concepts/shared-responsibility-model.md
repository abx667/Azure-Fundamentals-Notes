# 🔐 Shared Responsibility Model
 
The shared responsibility model defines how responsibilities are divided between the cloud provider and the customer.
 
---
 
## 🏢 On-Premise
You manage everything:
- Physical servers
- Networking
- OS
- Applications
 
---
 
## ☁️ IaaS (Infrastructure as a Service)
Azure manages:
- Hardware
- Networking
 
You manage:
- Operating System
- Applications
 
---
 
## ⚙️ PaaS (Platform as a Service)
Azure manages:
- Hardware
- OS
 
You manage:
- Applications
 
---
 
## 🧑‍💻 SaaS (Software as a Service)
Azure manages everything.
You just use the software.
