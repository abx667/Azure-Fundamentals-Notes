# ⚡ Serverless Computing
 
Serverless computing allows you to run applications without managing servers. The cloud provider automatically handles infrastructure, scaling, and maintenance.
 
---
 
## 🔑 Key Features
 
- No server management  
- Automatic scaling  
- Pay only for usage (per execution)  
- Event-driven execution  
 
---
 
## 🧑‍💻 Responsibility
 
Azure manages:
- Infrastructure  
- Server management  
- Scaling  
 
You manage:
- Application logic  
- Code  
 
---
 
## 📌 Examples
 
- Azure Functions  
- Azure Logic Apps  
 
---
 
## ✅ Advantages
 
- No need to manage servers  
- Scales automatically  
- Cost-effective for intermittent workloads  
 
---
 
## ⚠️ Challenges
 
- Costs may vary month-to-month  
- Cold start delays (initial response delay)  
 
---
 
## 🎯 Use Case
 
- Event-based processing (e.g., file upload triggers function)  
- Background jobs  
- APIs  
 
---
 
## 🧠 Quick Revision
 
- No server → fully managed
