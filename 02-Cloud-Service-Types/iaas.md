# 🏗️ Infrastructure as a Service (IaaS)
 
IaaS provides basic computing infrastructure such as virtual machines, storage, and networking over the cloud.
 
---
 
## 🔑 Key Features
 
- Virtual Machines (VMs)
- Storage
- Networking
- High flexibility and control
 
---
 
## 🧑‍💻 Responsibility
 
You manage:
- Operating System
- Applications
- Data
 
Azure manages:
- Hardware
- Physical network
- Data centers
 
---
 
## 📌 Example
 
Azure Virtual Machines
 
---
 
## 🎯 Use Case
 
- When you want full control over the environment  
- Migrating on-premise applications to cloud
